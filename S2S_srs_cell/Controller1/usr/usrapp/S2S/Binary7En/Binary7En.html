 Description and requirements
  
Title: Mathematical and Binary functions

General description :
  This library provides a functions for binary and mathematical
  calculations.

  1) transforms a binary to a decimal or vice-versa
  2) provides OR, XOR, AND operators for binary calculation
  3) Exponent Calculation

Hardware required :
  - Robot system    : no specific robot system
  - Peripherals     : no peripherals required

VAL3 options :
  - VAL3 version    : 7.*
  - VAL3 licenses   : no licenses required
  - VAL3 add-ons    : none

Library version     : 7.6.1.0

Public variables    : none

Included libraries  : none

Miscellaneous       : ---

Keywords            : calculation, Operator, And, Or, Xor, Decimal, binary,
                      function, word.

public And(string& x_sBinary1, string& x_sBinary2, string& x_sReturn[---]) 
  Description of program
    This program provides an And logical operator for 2 binary values.
  
       0 and 0 = 0
       0 and 1 = 0
       1 and 1 = 1
  
  Input parameters
    string x_sBinary1 :
      role : 1 st binary number
    string x_sBinary2 :
      role : 2 nd binary number
  
  Output parameters
    string x_sReturn :
      role : Result of calculation x_sBinary1 AND x_sBinary2
  
  Example
    And("10101011","00011011",sResult)
